Навыки разработки фронтенда

  • HTML

    HTML (HyperText Markup Language) является стандартным языком разметки для создания веб-страниц. Я владею основами HTML и умею создавать структуру веб-страницы.

    Пример проекта:

    В одном из моих проектов я разрабатывал лендинг-пейдж для стартапа. Используя HTML, я создавал различные секции страницы, включая заголовки, параграфы, списки и изображения. Также я применял атрибуты и теги для определения стилей и расположения элементов.

  • CSS

    CSS (Cascading Style Sheets) используется для стилизации веб-страниц. Я владею основами CSS и умею создавать привлекательный дизайн для веб-страниц.

    Пример проекта:

    В рамках одного проекта я создавал стили для блога. Я использовал CSS для определения цветов, шрифтов, отступов и размеров элементов. Также я применял анимации и переходы, чтобы сделать веб-страницу более динамичной и привлекательной.

  • JavaScript

    JavaScript - это язык программирования, который позволяет добавить интерактивность и функциональность на веб-страницы. Я владею основами JavaScript и умею использовать его для создания динамических веб-приложений.

    Пример проекта:

    В одном из моих проектов я разрабатывал калькулятор. Я использовал JavaScript для обработки событий, реализации логики калькулятора и взаимодействия с пользователем.
